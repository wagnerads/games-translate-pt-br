# games-translate-pt-br

Arquivos de tradução para os Jogos Big Pharma e Chocolatier.

A tradução do Big Pharma foi inspirada em: http://traducaobigpharma.blogspot.com.br/

## Como usar as traduções

### Big Pharma
1. Localize o arquivo `strings-en.data` na pasta do jogo
2. Substitua pelo arquivo `strings-en.data` deste repositório

### Chocolatier
1. Localize o arquivo `strings.xml` na pasta do jogo
2. Substitua pelo arquivo `strings.xml` deste repositório
